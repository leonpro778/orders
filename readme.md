# LS Orders

Simple application to manage orders in our company.

We use:
- [Laravel 5.8](https://laravel.com/docs/5.8)
- [MySQL 5.7](https://www.mysql.com/)
- [Bootstrap 4](https://getbootstrap.com/)
- [jQuery](https://jquery.com/)
